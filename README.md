# HappyFeet
**HappyFeet** is a full-stack e-commerce platform for footwear, built with the **MERN** stack **(MongoDB, Express.js, React.js, Node.js)**. It includes a frontend user interface, an admin dashboard, and a backend API with authentication, product management, and order handling.

# Features
## User (Frontend)

- Browse and search products
- View product details
- Add products to the cart and place orders
- Track orders and returns
- Responsive UI with modern design

## Admin Dashboard

- Add, update, or remove products
- Manage orders and returns
- User management
- Admin authentication

## Backend

- REST API built with Node.js and Express.js
- Image uploads handled via Cloudinary
- Secure environment-based configuration

## Tech Stack

- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
- **Cloud Services:** Cloudinary (for image uploads)
- **Version Control:** Git & GitHub
